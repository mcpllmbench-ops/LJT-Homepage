**Authors:** Jinghan Zhang, Shiqi Chen, Junteng Liu, Junxian He

**Abstract:** This paper presents novel approaches for composing parameter-efficient modules using arithmetic operations. We demonstrate how arithmetic compositions can maintain model performance while significantly reducing the number of trainable parameters.

**Keywords:** Parameter Efficiency, Model Composition, Arithmetic Operations, Efficient Training, Neural Architecture
